## 第三步：注册微信小程序

### 3.1 注册账号

1. **访问注册页面**
   - 打开：https://mp.weixin.qq.com
   - 点击 **"立即注册"**

2. **选择账号类型**
   - 选择 **"小程序"**

3. **填写注册信息**
   - 邮箱：填写您的邮箱地址
   - 密码：设置一个安全的密码
   - 验证码：输入图形验证码
   - 勾选同意协议
   - 点击 **"注册"**

4. **邮箱验证**
   - 打开您的邮箱
   - 找到微信团队发送的验证邮件
   - 点击邮件中的链接完成验证

5. **信息登记**
   - **主体类型**：选择 **"个人"**（个人学习使用）
   - **主体信息**：填写您的个人信息
   - **管理员信息**：用微信扫码验证
   - 点击 **"提交"**

### 3.2 获取 AppID

1. **登录小程序后台**
   - 访问：https://mp.weixin.qq.com
   - 使用注册的账号登录

2. **找到 AppID**
   - 在首页或 **"开发"** → **"开发设置"** 中
   - 复制 **AppID**（格式类似：wx1234567890abcdef）
   - 保存好这个 AppID，稍后会用到

### 3.3 配置服务器域名（可选）
如果以后需要连接后端服务器，在 **"开发"** → **"开发设置"** → **"服务器域名"** 中配置

### 6.4 常见错误处理

**错误 1：图片加载失败**
- **原因**：图片路径不存在或图片文件缺失
- **解决**：
  1. 在 `images/` 文件夹中添加占位图片
  2. 或者暂时注释掉图片相关代码
  3. 可以使用网络图片地址（需配置域名白名单）

**错误 2：Cannot read property 'xxx' of undefined**
- **原因**：数据未初始化或访问了不存在的属性
- **解决**：
  1. 检查 `app.js` 中的 `globalData` 是否正确
  2. 在 `onLoad` 中确保数据已加载

**错误 3：navigateTo 失败**
- **原因**：页面路径错误或未在 `app.json` 中配置
- **解决**：检查 `app.json` 的 `pages` 配置

**错误 4：tabBar 不显示**
- **原因**：图标路径错误或图标文件缺失
- **解决**：
  1. 临时删除 `app.json` 中的 `iconPath` 配置
  2. 或添加正确的图标文件

## 常见问题解答

### Q1: 我没有编程基础，能完成吗？
**A:** 可以！只要按照本指南逐步操作，每一步都有详细说明。遇到问题可以先查看常见错误处理部分。

### Q2: 必须注册小程序账号吗？
**A:** 不是必须的。在导入项目时可以选择 **"测试号"**，大部分功能都可以正常使用，但无法发布。

### Q3: 图片资源去哪里找？
**A:** 
- 使用免费图库：Unsplash、Pexels
- 使用占位图服务：https://placeholder.com/
- 自己拍摄或设计
- 使用网络图片（需配置域名）

### Q4: 如何修改菜品数据？
**A:** 编辑 `app.js` 文件，修改 `globalData.menuData` 数组：
```javascript
menuData: [
  {
    id: 1,
    name: "分类名称",
    foods: [
      { 
        id: 101, 
        name: "菜品名称", 
        price: 18, 
        image: "/images/food.jpg", 
        desc: "菜品描述", 
        sales: 100 
      }
    ]
  }
]
```

### Q5: 如何修改主题颜色？
**A:** 修改以下文件：
- **导航栏颜色**：`app.json` 中的 `navigationBarBackgroundColor`
- **主色调**：全局搜索 `#ff6b6b`，替换为您想要的颜色

### Q6: 可以添加支付功能吗？
**A:** 可以，但需要：
1. 企业主体的小程序账号
2. 开通微信支付商户号
3. 后端服务器处理支付逻辑
4. 配置支付参数和回调

### Q7: 如何部署到线上？
**A:** 
1. 确保使用的是正式 AppID（非测试号）
2. 点击开发者工具的 **"上传"** 按钮
3. 填写版本号和描述
4. 登录小程序后台，提交审核
5. 审核通过后发布

### Q8: 数据会丢失吗？
**A:** 
- 购物车数据：存储在本地，卸载小程序后会丢失
- 订单数据：存储在本地，卸载小程序后会丢失
- 如需永久保存，需要连接后端服务器

### Q9: 如何连接后端服务器？
**A:** 
1. 准备后端 API（Node.js、Python、Java 等）
2. 配置 HTTPS 域名（必须）
3. 在小程序后台配置服务器域名
4. 使用 `wx.request()` 调用 API

### Q10: 遇到其他问题怎么办？
**A:** 
1. 查看开发者工具的控制台错误信息
2. 搜索微信小程序官方文档
3. 在 GitHub 项目页面提 Issue
4. 访问微信开放社区：https://developers.weixin.qq.com/community/
