## 第二步：完善小程序基本信息

### 2.1 设置小程序名称

**路径**：设置 → 基本设置 → 小程序名称

**操作**：
1. 点击"修改"
2. 输入名称：`蓓蓓的餐厅`
3. 微信扫码确认
4. 等待审核（通常 7 个工作日内）

**注意**：
- 个人主体一年只能修改 2 次
- 企业主体一年只能修改 3 次
- 名称不能侵权或违规

### 2.2 上传小程序头像

**路径**：设置 → 基本设置 → 小程序头像

**要求**：
- 格式：JPG/PNG
- 大小：不超过 2MB
- 尺寸：建议 144x144px
- 内容：清晰、美观、与餐厅相关

**建议**：
- 可以使用餐厅 logo
- 或者使用美食相关的图标
- 保持简洁、易识别

### 2.3 填写小程序介绍

**路径**：设置 → 基本设置 → 小程序介绍

**示例文案**：
```
蓓蓓的餐厅，位于深圳市福永街道聚福园，提供在线点餐服务。
精选优质食材，用心烹饪每一道菜，为您带来美味的用餐体验。
```

**要求**：
- 不超过 120 字
- 准确描述小程序功能
- 不能有违规内容

### 2.4 设置服务类目

**路径**：设置 → 基本设置 → 服务类目

**推荐类目**：
1. **餐饮** → **点餐平台**
   - 适用于提供线上点餐服务
   - 可能需要提供：营业执照、食品经营许可证

2. **生活服务** → **餐饮**
   - 适用于餐厅信息展示
   - 要求相对宽松

**个人主体限制**：
- 个人主体能选择的类目有限
- 餐饮相关类目通常需要企业资质
- 可能只能选择"工具" → "信息查询"等通用类目

**如何选择**：
```
1. 点击"添加"
2. 选择类目
3. 上传资质证明（如需要）
4. 等待审核
```

## 第四步：上传代码

### 4.1 在微信开发者工具中上传

**操作步骤**：

1. **打开项目**
   ```
   路径：F:\Code_test\WeChat_ordering_minipro-main
   AppID：wx22a1234a7f62cfe9
   ```

2. **编译并测试**
   - 点击"编译"确保无错误
   - 测试所有功能正常
   - 检查图片都能正常显示

3. **点击"上传"按钮**
   - 位置：顶部工具栏，"预览"按钮旁边
   - 快捷键：Ctrl + Shift + U

4. **填写版本信息**
   ```
   版本号：1.0.0
   项目备注：蓓蓓的餐厅点餐小程序首个版本
   ```

5. **点击"上传"**
   - 等待上传完成
   - 会显示上传成功的提示

**注意事项**：
- 确保使用正式 AppID（非测试号）
- 代码包大小限制：单个分包不超过 2MB，总大小不超过 20MB
- 当前项目大小约 100KB，完全符合要求

### 4.2 常见上传错误

**错误 1：AppID 无效**
- 原因：使用了测试号
- 解决：使用正式注册的 AppID

**错误 2：未配置合法域名**
- 原因：代码中有网络请求，但未配置域名
- 解决：在后台配置服务器域名，或删除网络请求代码

**错误 3：代码包过大**
- 原因：包含了过大的图片或文件
- 解决：压缩图片，删除无用文件

- 建议提供自己和朋友的微信号

**3. 补充说明（重要）**

示例文案：
```

本小程序为餐厅点餐系统，提供以下功能：
```

- 路径：管理 → 版本管理

## 第六步：处理审核结果

### 6.1 审核通过

**操作**：
1. 收到审核通过通知（微信消息/邮件）
2. 登录小程序后台
3. 路径：管理 → 版本管理 → 审核版本
4. 点击"提交发布"
5. 等待发布完成（几分钟）

**发布成功后**：
- 用户可以通过搜索找到小程序
- 可以生成小程序码用于推广
- 可以在微信聊天中分享

### 6.2 审核不通过

**常见原因**：

1. **类目不符**
   - 问题：选择的类目与实际功能不匹配
   - 解决：修改服务类目，重新提交

2. **缺少资质**
   - 问题：餐饮类目需要营业执照、食品经营许可证
   - 解决：上传相关资质，或改用个人可用的类目

3. **功能不完整**
   - 问题：审核人员无法正常使用某些功能
   - 解决：优化代码，完善功能，重新上传

4. **内容违规**
   - 问题：包含违规文字、图片
   - 解决：修改内容，重新提交

5. **缺少必要说明**
   - 问题：审核人员不理解如何使用
   - 解决：补充详细的测试说明

**处理流程**：
```
1. 查看驳回原因
2. 修改代码或补充说明
3. 重新上传版本
4. 再次提交审核
```

## 💡 常见问题

### Q1: 个人可以发布餐饮小程序吗？
A: 可以，但类目选择有限制。建议选择"工具"类目，作为餐厅内部点餐工具。

### Q2: 审核一般要多久？
A: 通常 1-7 个工作日，首次可能稍长。

### Q3: 审核不通过怎么办？
A: 根据驳回原因修改，可以多次提交审核，无次数限制。

### Q4: 发布后可以修改吗？
A: 可以，上传新版本重新审核即可。

### Q5: 如何让更多人看到？
A: 线下张贴小程序码，线上分享，优化小程序名称和介绍。

### Q6: 需要服务器吗？
A: 当前项目不需要，数据存本地。如需在线支付等功能，需要后端服务器。

### Q7: 个人小程序可以盈利吗？
A: 个人主体不能接入微信支付，无法直接收款。

### Q8: 可以改名吗？
A: 可以，但个人主体一年只能改 2 次。
