# 🚀 5分钟快速开始

如果您想最快速度看到效果，按照以下步骤操作：

## 步骤1：创建 GitHub 仓库（2分钟）

1. 登录 https://github.com
2. 点击右上角 "+" → "New repository"
3. 填写：
   - Repository name: `wechat-restaurant-ordering`
   - 勾选 "Add a README file"
   - 点击 "Create repository"

## 步骤2：上传代码（2分钟）

1. 点击 "Add file" → "Upload files"
2. 拖拽所有项目文件
3. 点击 "Commit changes"

## 步骤3：运行小程序（1分钟）

1. 下载并安装微信开发者工具
2. 扫码登录
3. 导入项目：
   - 项目目录：选择下载的项目文件夹
   - AppID：选择"测试号"
   - 点击"导入"
4. 点击"编译"按钮

✅ 完成！现在您应该能看到运行的小程序了。
